# プロジェクト概要：Customer Portal Sync (CPS)
**受託内容**: ソースコード納品（バックエンドAPI + フロントエンドUI）

## 1. 開発フロー
本プロジェクトでは、Gemini Skills エコシステムを活用し、以下の自動化プロセスで開発を推進する。

| フェーズ | 使用スキル | 成果物 |
| :--- | :--- | :--- |
| **立上げ** | `boilerplate-genie` | プロジェクト雛形、CI/CD構成 |
| **要件定義** | `requirements-wizard` | IPA準拠 要件定義書 |
| **インフラ設計** | `environment-provisioner` | Terraform, Dockerfile |
| **実装・検証** | `test-suite-architect` | Jest/Pytest テストコード |
| **品質・安全監査** | `security-scanner`, `project-health-check` | セキュリティスキャン報告書、健全性スコア |
| **納品準備** | `pr-architect`, `ppt-artisan` | 納品レポート、API仕様書 |
