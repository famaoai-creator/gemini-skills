# 画面レイアウト設計：ダッシュボード

## 1. 構成要素
- **Header**: ロゴ、ユーザーアイコン、通知ベル。
- **Sidebar**: プロジェクト一覧、期限間近のタスク、カレンダー。
- **Main Content**: 
    - **Summary Cards**: 「完了済み」「進行中」「期限切れ」の数値。
    - **Active Tasks**: 直近 5 件のタスクリスト（カード形式）。

## 2. デザイン方針 (`ux-auditor`)
- **Fitts's Law**: 「新規タスク作成」ボタンは右下に固定配置（Floating Action Button）し、親指で押しやすくする。
- **Color**: 進行中＝青、期限切れ＝赤、完了＝緑で状態を直感的に識別。
