# UI/UX 監査・デザイン方針書 (GAI-OVS)
**準拠**: `ux-auditor` ヘリスティックス分析

## 1. 視覚的階層 (Visual Hierarchy)
- **カードレイアウト**: 各顧客情報は独立したカードで表示。
- **コントラスト**: 背景色（#F9F9F9）と文字色（#333333）のコントラスト比を 4.5:1 以上に維持。

## 2. 認知負荷の低減 (Cognitive Load)
- **検索**: 入力と同時にリアルタイムでフィルタリングを行い、ユーザーの記憶負荷を軽減。
- **1画面1アクション**: 複雑なサイドバーを廃止し、中央の検索窓に集中させる。

## 3. アクセシビリティ (WCAG 2.1)
- すべての対話型要素には適切な `aria-label` を付与。
- キーボードのみでの操作をサポート。
