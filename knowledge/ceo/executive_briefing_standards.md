# Executive Briefing Standards

## 1. The "CEO Rule of 3"
技術的な詳細に関わらず、報告は常に以下の 3 点に集約すること。

1.  **Business Impact**: どの KPI（コスト、時間、リスク）に影響したか。
2.  **Current Status**: 信号機形式（緑・黄・赤）による現状把握。
3.  **Required Decision**: 経営者による判断や承認が必要な事項は何か。

## 2. Terminology Translation
| Technical Term | Executive Language |
| :--- | :--- |
| Latency Regression | Process Slowdown (Risk of inefficiency) |
| SLO Breach | Reliability Commitment Risk |
| MD5 Hash Validation | Data Integrity Assurance |
| Chaos Monkey | Resilience Stress Testing |
