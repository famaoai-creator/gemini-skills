# macOS 音声コントロール ＆ 音声入力活用ガイド

macOS 標準機能を活用し、キーボードを使わずに Gemini CLI を操作するためのセットアップ手順。

## 1. 音声入力 (Dictation) の最適化
短文のコマンド入力に最適です。

### 設定手順
1. **システム設定 > キーボード > 音声入力** をオンにする。
2. **ショートカット** を「Fnキー 2回」などに設定。
3. **言語**: 日本語だけでなく「英語 (米国)」も追加することを強く推奨（技術用語の認識精度が段違いです）。

### CLIでの活用
- ターミナルで `Fn` x 2 を押し、「Run security scan」と話しかけるだけで入力可能。

## 2. 音声コントロール (Voice Control)
完全ハンズフリー操作が必要な場合に利用します。

### 設定手順
1. **システム設定 > アクセシビリティ > 音声コントロール** をオンにする。
2. **カスタムコマンド** の作成:
    - **発話**: "Hey Gemini"
    - **アクション**: テキストを貼り付け "gemini run mission-control" して Enter を押す。

## 3. `voice-command-listener` との使い分け
- **OS標準機能**: テキスト入力の代替。プロンプトを「書く」のが面倒な時に使う。
- **Skill (`voice-command-listener`)**: 録音データを解析させたい時、または会議の議事録をそのままコマンド化したい時に使う。
