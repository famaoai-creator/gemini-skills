# プロフェッショナル・エクセル設計書テーマガイド

高品質な設計書（エクセル）を出力するための、レイアウトとビジュアルの標準基準。

## 1. ヘッダー・メタデータ (共通フレーム)

全シートの最上部（1-3行目）に以下の情報を配置し、ドキュメントの追跡可能性を確保する。

| カラム               | ラベル例                   | 役割             |
| :------------------- | :------------------------- | :--------------- |
| **プロジェクト情報** | プロジェクト名, システム名 | 所属の明確化     |
| **工程情報**         | 工程, 成果物名             | 文書の位置づけ   |
| **管理情報**         | 作成, 変更, 確認           | 責任と履歴の明示 |

## 2. 視覚的スタイル

- **フォント**: ＭＳ Ｐゴシック または 遊ゴシック。
- **配色**:
  - 見出し背景: 濃紺（#000080）または 濃緑（#006400）。
  - 文字色: 白（#FFFFFF）。
  - 交互行背景: 薄いグレー（#F2F2F2）。
- **罫線**:
  - 外枠: 太線。
  - 内枠: 細線（格子）。

## 3. シート構造

- **章立て**: `1.`, `1.1.`, `1.1.1.` の形式で階層化し、インデントで視覚的に表現する。
- **図表の配置**: 表形式（グリッド）を基本とし、複雑な関係図は `diagram-renderer` で生成した画像を適切なセル範囲に挿入する。

## 4. Excel-Artisan への指示

エクセル生成スキルを利用する際は、以下の JSON 構造でスタイルを指定することを推奨する。

```json
{
  "header": { "bg": "#006400", "fontColor": "#FFFFFF", "bold": true },
  "grid": { "border": "thin", "borderColor": "#CCCCCC" }
}
```
