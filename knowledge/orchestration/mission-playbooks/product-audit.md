# ミッション・プレイブック：プロダクト健全性 ＆ リリース監査

本プレイブックは、既存のプロジェクトが本番レベルの品質基準を満たしているかを客観的に評価し、納品責任を果たすための「勝利条件」を定義する。

## 1. 必須スキルセット
- `project-health-check`
- `security-scanner`
- `ux-auditor`
- `pmo-governance-lead`

## 2. 勝利条件
- [ ] **全方位監査**: 技術・セキュリティ・UX の 3軸 で PASS 判定が得られていること。
- [ ] **定量的証明**: カバレッジ 90% 以上、重大な脆弱性 0 件。
- [ ] **ビジネス価値の紐付け**: 品質スコアが「保守コストの削減」にどう寄与するかを `business-impact-analyzer` で解説。

## 3. アウトプット
- `docs/audit_report.md` (詳細証跡)
- `delivery_presentation.pptx` (ステークホルダー向け)
