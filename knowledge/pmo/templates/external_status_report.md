# 対外報告用ステータスレポート・テンプレート

外部のPMOや顧客へ提出する、プロジェクトの進捗と信頼性を証明するための構成。

## 構成案
1. **ハイライト (High-level Status)**: 
   - 信号色（青：順調、黄：注意、赤：問題あり）による一目での状況報告。
2. **マイルストーン進捗**: 
   - 計画 vs 実績。
3. **品質・セキュリティエビデンス**: 
   - `security-scanner` と `test-genie` の最新サマリー。
4. **リスクと対策**: 
   - 現在検知されているリスクと、それに対する具体的な Mitigation Plan（緩和策）。
5. **次期の予定**: 
   - 直近 2週間 の注力項目。
