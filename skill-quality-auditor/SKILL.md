# Skill Quality Auditor

This skill maintains the high standard of the monorepo itself.

## Capabilities

### 1. Documentation Audit
- Checks if `SKILL.md` has required sections (Capabilities, Usage, etc.).
- Validates that descriptions are concise.

### 2. Implementation Check
- Verifies that referenced scripts actually exist.
- Checks if the skill is correctly installed in the environment.

## Usage
- "Audit all skills in this monorepo and report those with missing documentation or broken scripts."

## Knowledge Protocol
- This skill adheres to the `knowledge/orchestration/knowledge-protocol.md`. It automatically integrates Public, Confidential (Company/Client), and Personal knowledge tiers, prioritizing the most specific secrets while ensuring no leaks to public outputs.
